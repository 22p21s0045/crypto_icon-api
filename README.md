
# Crypto icon api

## API Reference





#### Get icon svg

```http
  GET /api/icon/${symbol}
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `symbol`      | `string` | **Required**.  code of item to fetch |
